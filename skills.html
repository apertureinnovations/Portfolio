Other Languages

Python

This is the language i started coding in, and as a result i know it better than all others. It is quite similar to JS on the way it handles classes, objects, statements and loops, but varies quite a bit when it comes to the use case; if python is ever used on a website, it would be on the backend, like PHP, with a module like django or flask.

PHP

This is a language i plan to learn in the future, leading towards my goal of being a full stack developer. It handles the back end of websites. It can generate dynamic page content and can w/r files on the server, it allows for use of cookies, can modify data in SQL, it can control user-access levels and can encrypt data. It's a very powerful tool.

Skillsets

HTML5

Often Called the structure of websites, HTML is a key skill in any developers aresenal, and is what the front end builds upon.

CSS3

This is how we make our sites look good! CSS is the core component in web design, and as such is used just as much as html, if not more.

JScript

This is how we control the behaviour of our sites; It allows us to control aspects of the site, allowing for a responsive experience.

Knowing these three is essential to any Front-End developers' toolkit, but thats not all! We also need to know how to manipulate libaries and frameworks to save time whilst enchancing the user experience.

Bootstrap

Bootstrap is a CSS libary with JS elements originating from jQuery, and is extremely useful in putting together core elements of a site together fast; It allows for easier deployment of sites but lacks the finetuning that coding in pure CSS w/ JS has, but makes up for it in the quality of the classes created by Bootstrap.

jQuery

jQuery is a framework for JS that essentially allows for shorthand JS, and is a very powerful tool which streamlines the JS process. This allows for faster deployment without sacrificing quality or features, which is why it is the most used JS framework and the most extendable. Many companies use jQuery for this very reason.
